В данном архиве представлен отчёт по модулю B. Рабочее место 2. Конкурсант Сиотнаов Алексей Максивович.
Состоит из:
	1-Код в Report_M4.html для проверки.
	2-Код в Report_M4.ipynb - исходник для проверки.
	3-Папка API, в котором присутствуют файлы для работы с моделью через графический интерфейс.
	4-Папка Фото к отчёту, там расположены фото, которые находятся в отчете Report_M4.html

Иснстукция:
- Необходимо запустить app.py, который находится в папке API.
- Далее перейти в любой браузер и перейти по ссылке http://127.0.0.1:500/
- После чего можно вставить ссылку в необходимое для этого места, нажать на кнопку прогноз.
- Наконец необходимо подождать пока модель сделает прогноз и выведет его вам.
